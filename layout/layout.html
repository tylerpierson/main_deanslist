USER PAGES

  • Admin page
    • Super capabilities
    • Able to access all student data
    • Capable of adding/editing/removing teachers/students/parents
    • Able to create teacher groups
    • Can decide what student information is public and private throughout groups
    • Able to view data in an "overall" graph
  • Teacher Page
    • Able to edit teacher(self) information
    • Able to view (but not edit) student information that is made public within teacher groups
    • Able to add/edit/remove classes/students
    • Able to add/edit/remove assignments/assessments
      • Able to add an overarching "Target" to the assessments that will help determine which content areas students are most successful in
      • Prompted to select grading scale and content area when creating assignment
      • Each assigned question will have a "Target" that will determine which groups the students are placed in
      • Students will be placed in groups based on common areas of deficiencies or strengths with teachers that showed the most success in teaching the content
    • Able to input and edit student grades
    • Able to input student DRA/Lexile levels
      • Students can be grouped by level
      • Goals can be set with percentage of success
      • Page will show how many students until goal is met
  • Parent Page
    • Able to view individual student pages with scores
    • Able to make comments and message the student's teacher if capabilities are enabled
    • Can edit personal information
      • Name
      • Address
      • Phone Number
    • Able to view student areas of definciencies and areas of strength
  • Student Page
    • Able to view individual student pages with scores and areas of growth/decline
    • Students will have the ability to print certificates when successful in certain content areas on ASSESSMENTS
